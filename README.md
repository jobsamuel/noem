# noem

A demo API REST that uses Node, Express and MongoDB.

# usage

### Pre-requisites

 - [NodeJS](http://nodejs.org/)
 - [MongoDB](http://www.mongodb.org/downloads)

### Setup

```
$ git clone git@github.com:jobsamuel/noem.git
$ npm install
$ node app/app.js
```
